## WeChatMsg_Lite

在 WeChatMsg 基础上修改的轻量化版本，主要用于从 windows 电脑端微信导出聊天记录文件。



### 微信版本要求

“在微信 Windows 4.0.3.36 & macOS 4.0.3.80 以上的版本中，密钥信息不再长时间保留在内存数据中，所以无法通过当前方案获取密钥。” 经过测试，虽然 WeChatMsg 代码进行了微信4.x版本的适配，但是 4.0.3.36 等较新的版本无法获取微信昵称等信息。

因此**推荐使用 4.0.3.19 版本的电脑端微信**。安装包已附加在项目根目录。

如需使用其他版本的电脑端微信，请自行对代码进行修改。



### 代码功能

1. 解密并读取 Windows 本地微信数据库；
2. 查看联系人和群聊；
3. 导出聊天记录为 CSV / HTML / TXT / DOCX / Markdown / XLSX；
4. 不包含 GUI；
5. 不包含 AI 功能。
